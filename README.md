
# EventPhotoShareApp
Mobile application written in React Native.

## Getting Started

1. clone the repo   

```
git clone https://github.com/LoureiroPT/event-photo-share-app.git
```

2. install dependencies   
```
npm i
```
or
```
yarn
```

3. link dependencies  
```
react-native link
```

4. start in ios or android   

```
react-native run-ios
```
or
```
react-native run-android
```
